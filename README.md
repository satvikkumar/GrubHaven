# GrubHaven 
Working Repository of GrubHaven for a Software Engineering Project

# Usage
## Server
 - Install NodeJS & MongoDB
 - Start the mongodb service
 - clone this repository `git clone https://github.com/satvikkumar/GrubHaven`
 - `cd GrubHaven` 
 - `npm install`
 - `npm run debug`
